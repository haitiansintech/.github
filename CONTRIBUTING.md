# Contributing to Haitians in Tech

We’re glad you’re here.

This guide applies to all Haitians in Tech open-source projects.

## Step 1: Pick a Project
Choose a project that interests you:
- Haiti City Portal
- VWA

## Step 2: Pick an Issue
Look for issues labeled:
- good first issue
- help wanted
- documentation
- frontend
- backend

## Step 3: Claim the Issue
Comment on the issue and say you want to work on it.
A maintainer will assign it to you.

## Step 4: Make Your Changes
- Fork the repository
- Create a new branch
- Submit a pull request

## How We Work
- Be kind and respectful
- Ask questions early
- Keep PRs small and focused

## Code of Conduct
We follow the Contributor Covenant.
